## 介绍
SlideExtractor.WPF 是一个用于从幻灯片中提取内容的应用程序。它支持多种幻灯片格式，并能高效地提取文本、图像和其他嵌入的媒体文件。此应用程序旨在帮助用户快速获取幻灯片中的信息，以便于后续的编辑和使用。

## 特性
- 支持多种幻灯片格式
- 高效提取文本、图像和媒体文件
- 友好的用户界面
- 支持批量处理幻灯片文件

## 安装
请确保您的计算机上已安装 [.NET 6.0](https://dotnet.microsoft.com/download/dotnet/6.0) 或更高版本的运行时。

下载最新版本的 SlideExtractor.WPF 安装程序，并按照提示完成安装。

## 使用方法
1. 启动 SlideExtractor.WPF 应用程序。
2. 通过拖放或点击“添加文件”按钮导入幻灯片文件。
3. 选择输出目录。
4. 点击“提取”按钮开始提取过程。
5. 提取完成后，您可以在输出目录中找到提取的文件。
